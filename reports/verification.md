## 分析精度の改善提案

### → 銘柄選定・仮説チームへ
担当: **シミュレーション追跡・検証チーム → 銘柄選定・仮説チーム**

#### 【提案①】Aランク選定基準への「イベントリスク係数」の導入

銘柄選定チームのレポートでは、アドバンテストについて3/30・4/1・4/4の3イベントが来週に集中することが適切に指摘されている。これを**定性的な注意事項**にとどめず、以下のように**定量的なフィルタ**として組み込むことを提案する：

```
イベントリスク評点（0〜3点）
　1点: 重大イベントが週内に1件
　2点: 重大イベントが週内に2件
　3点: 重大イベントが週内に3件以上
→ 2点以上の場合、Aランクを「A-（条件付きA）」に格下げし、
　 エントリーサイズを通常の50%に制限する
```

#### 【提案②】ミネルヴィニ基準の数値チェックリストの標準化

SUMCOのレポートに「ミネルヴィニ基準（52週安値比+30%以上、52週高値から25%以内等）」の記述があるが、**「等」の部分を明示化**することで追跡チームとの認識ズレを防ぐ。

推奨チェックリスト（ミネルヴィニ準拠）：

| 条件 | 基準 | 確認 |
|------|------|------|
| 52週安値比 | +30%以上 | □ |
| 52週高値比 | -25%以内 | □ |
| 200日MA | 現値が上位 | □ |
| 50日MA > 200日MA | 確認 | □ |
| 直近四半期EPS成長 | +25%以上 | □ |
| 直近四半期売上成長 | +25%以上 | □ |
| 株価が52週高値付近 | 高値-15%以内 | □ |

#### 【提案③】RS（相対強度）スコアの数値明記

仮説チームのレポートにRS値の言及が見られない。Gemini情報でも指摘されているとおり、**RSIの期間設定と閾値の調整**（ボラティリティ連動）が精度向上に有効であるため、各銘柄に対して以下を明記することを依頼したい：

- RS順位（市場全体比・セクター内比）
- 使用RSI期間（14日 or 調整値）
- 直近RSI水準（過熱・調整局面の判断根拠として）

### → 投資戦略チームへ
担当: **シミュレーション追跡・検証チーム → 投資戦略チーム**

#### 【提案①】「Defendフェーズ」における部分エントリールールの明文化

現状の戦略チームレポートでは「現金保有優先」が鉄則とされているが、**「防御フェーズ中に例外的にエントリーを許容するケース」**が明文化されていない。追跡チームとして以下の基準設定を提案する：

```
Defendフェーズ中のエントリー許容条件（案）
　① 銘柄のRS順位が上位10%以内（市場全体に対して相対的に強い）
　② エントリー可能資金の上限を通常の30%以内に制限
　③ ロスカット基準を通常(-8%)から厳格化（-5%）
　④ 目標①は通常(+25%)を縮小し、+10〜+15%で半分利確
```

戦略チームが「信用買い残5.8兆円超（約20年ぶり高水準）」を重大リスクとして指摘している。これは**追跡チームの損切り判断にも直接影響する**ため、毎週のレポートに以下を固定項目として追加することを依頼したい：

| 指標 | 今週値 | 前週比 | 警戒水準 |
|------|--------|--------|---------|
| 信用買い残（兆円） | 5.8 | - | 5.5兆円超で警戒 |
| 信用倍率 | - | - | 6倍超で警戒 |

#### 【提案③】S&P500の200日MA位置と日本株エントリーの相関ルール化

戦略チームレポートに「S&P500が200日MAを下方ブレイク済み」とある。過去の相関分析から、**米国主要指数が200日MA下の局面での日本株個別銘柄エントリー勝率**を将来的に集計・フィードバックすることを提案する。初期仮説として以下を設定：

     モメンタム戦略の勝率が通常時比-15〜20%低下する
→ 10件以上のサンプル蓄積後に検証予定
