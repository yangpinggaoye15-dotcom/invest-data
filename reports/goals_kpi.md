## 全体方針

### 運用前提
- **運用資産**: 200万円（2026年3月時点）
- **戦略**: ミネルヴィニStage-2（段階的にポジションサイズ拡大）
- **最終意思決定者**: オーナー（人間）
- **統括マネージャー**: 情報整理・提案・実行管理・PDCA推進（意思決定以外すべて）

### 最終ビジョン
**3年後に資産1億円以上**

### 段階的目標（四半期ごとに実績ベースで引き上げ）

| フェーズ | 期間 | 開始資産 | 目標資産 | 目標リターン | ポジション上限 | トレード頻度 |

| Phase 1→2 | 勝率50%以上 + 月次損益2ヶ月連続プラス + 最大DD-10%以内 | Phase 1を継続、目標据え置き |
| Phase 2→3 | 勝率55%以上 + PF1.5以上 + 最大DD-10%以内 | Phase 2を継続 |
| Phase 3→4 | 勝率55%以上 + PF2.0以上 + 最大DD-10%以内 | Phase 3を継続 |

### Phase 1 達成ロジック（検証対象）

```
開始資産: 200万円 → 目標: 300万円（+100万円）
期間: 3ヶ月（約60営業日）

1銘柄30万円 × 月4回 = 月12回トレード（複利で再投入）
勝率50%、RR比3.0の場合:
  勝ち: +9万円（+30%）  負け: -2.4万円（-8%）
  月12回 → 6勝6敗 = +9×6 - 2.4×6 = +39.6万円/月
  3ヶ月 → +100万円超（複利効果で加速）

必要な変更点:
  - RR比を2.0→3.0に引き上げ（目標①を+30%に変更）
  - トレード頻度を月2回→月4回に増加
  - Steady/Attack局面では積極的にエントリー
```

## 統括マネージャーの役割

| 項目 | 内容 |
|------|------|
| **やること** | 全チームの作業管理、情報集約、KPI監視、PDCA推進、改善提案の取りまとめ、レポート統括 |

## PDCAサイクル

### 回転スケジュール

| サイクル | Plan | Do | Check | Act |
|---------|------|-----|-------|-----|
| **日次** | 当日のアクションプラン策定 | データ収集・分析・シミュレーション実行 | デイリーレポートで結果確認 | 翌日の計画に即反映 |
| **週次** | 週間目標設定（翌週イベント対策） | 5営業日のトレード/シミュレーション | KPIダッシュボード確認・差異分析 | 分析手法・パラメータ調整 |
| **月次** | KPI目標値の見直し・戦略修正 | 月間のトレード実行 | 月次KPI集計・学習パターン棚卸 | チーム体制・ルール改定 |
| **四半期** | 年間目標への進捗評価・戦略大幅見直し | 四半期のトレード実行 | 全KPI総括・戦略有効性検証 | 運用ルール・目標値の全面改定 |

### 日次PDCAの具体フロー

```
[Plan] 朝 ─────────────────────────────────────────────────
  統括マネージャー: 前日の市場結果を踏まえ、本日のアクションプランを策定
  → オーナーに判断が必要な事項を提示

[Do] 日中 ──────────────────────────────────────────────────
  情報収集: 市場データ・ニュース収集
  分析: スクリーニング・銘柄評価
  リスク管理: リスク要因チェック
  投資戦略: エントリー候補の設計
  検証: シミュレーション追跡・記録

[Check] 夕方 ───────────────────────────────────────────────
  検証チーム: 予測vs実績の差異確認
  統括マネージャー: デイリーレポート集約・KPI更新

[Act] レポート作成 ─────────────────────────────────────────
  差異原因の特定 → 翌日の分析手法に即反映
  学習パターン更新 → 翌日のシミュレーションに反映
```

#### Phase 1 達成へのアクションプラン

**目標: 200万→300万円（+100万円 / 3ヶ月）**

1. **RR比3.0を狙う設計に変更**
   - 1銘柄30万円、損切り-8%（-2.4万円）、目標+30%（+9万円）
   - 月4回 × 勝率50% → 2勝2敗 = +9×2 - 2.4×2 = **+13.2万円/月**
   - 複利で資産増加分を再投入 → 3ヶ月で+100万円ペース

2. **Defend局面はシミュレーションのみ、Steady/Attackで攻める**
   - 無駄な負けを減らし、好環境で集中的にトレード

3. **週次PDCAで軌道修正**
   - 月+33万円ペースを維持しているか毎週確認
   - 未達時は即座にエントリー基準・RR設定を調整

4. **Phase 2移行条件を満たすことが最優先**
   - 勝率50%以上 + 月次2ヶ月連続プラス + DD-10%以内
   - 条件クリアでポジションサイズを段階的に拡大
